# 🎙️ 뉴스공장 인터뷰 전문 요약 서비스 백엔드 (News Factory Transcript Summary Backend)

이 프로젝트는 **'김어준의 겸손은힘들다 뉴스공장'** 공식 홈페이지의 인터뷰 전문을 자동으로 크롤링하여 핵심 내용을 요약하고 캐싱하는 FastAPI 기반의 백엔드 API 서비스입니다.

---

## 🛠️ Tech Stack (기술 스택)

- **Language**: Python 3.12+
- **Framework**: FastAPI
- **Package Manager**: `uv` (Fast & Modern package manager)
- **Database (Cache)**: Supabase (PostgreSQL)
- **AI Core**: Gemini API (`gemini-3.1-flash-lite`) using `google-genai` SDK
- **Web Crawler**: HTTPX & BeautifulSoup4

---

## ✨ 핵심 기능

1. **자동 크롤러**:
   - `https://humblefactory.co.kr/category/transcript/` 메인 카테고리 페이지를 파싱하여 오늘 자(KST 기준) 혹은 특정 날짜에 업로드된 인터뷰 제목과 상세 링크를 수집합니다.
   - 각 인터뷰 본문 상세 링크로 진입하여 전문 텍스트를 실시간으로 크롤링합니다.
2. **다중 게시글 중복 캐싱 로직**:
   - 인터뷰 전문의 상세 URL(`content_url`)을 고유 키로 활용하여 DB 중복 체크를 수행합니다.
   - 이미 요약된 이력이 있는 글은 **Supabase**에서 즉시 조회하여 반환(Cache Hit)하므로 API 응답 속도가 대폭 향상되고 AI 토큰 비용을 방지합니다.
   - 캐시에 없는 새로운 게시글만 상세 본문을 긁어와 요약을 진행하고 DB에 저장(Cache Miss)합니다.
3. **빈티지 톤앤매너 요약**:
   - Google의 최신 `gemini-3.1-flash-lite` 모델을 활용하여, 인터뷰 내용을 차분하고 정갈한 빈티지 톤으로 **핵심 내용 3줄 요약** 및 **주요 참석자 정보**를 정갈하게 제공합니다.

## 🗄️ Supabase 데이터베이스 설정

Supabase 프로젝트 생성 후, SQL Editor에서 아래 DDL 쿼리를 실행하여 테이블 및 인덱스를 구축해 주어야 합니다.

```sql
CREATE TABLE IF NOT EXISTS summaries (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    title TEXT NOT NULL,
    content_url TEXT UNIQUE NOT NULL,
    summary_text TEXT NOT NULL,
    target_date DATE NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- 빠른 조회를 위한 URL 및 날짜 인덱스 생성
CREATE INDEX IF NOT EXISTS idx_summaries_content_url ON summaries(content_url);
CREATE INDEX IF NOT EXISTS idx_summaries_target_date ON summaries(target_date);
```

## ⚙️ 로컬 개발 환경 구축

### 1. 패키지 설치 (`uv` 사용)
`uv`가 설치되어 있지 않다면 먼저 설치한 후 진행해 주세요.
```bash
# 의존성 패키지 동기화 및 가상환경 설정
uv sync
```

### 2. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env` 파일을 작성하고 필요한 API 키를 채워 넣습니다.
```env
SUPABASE_PROJECT_URL=your_supabase_project_url
SUPABASE_SECRET_KEY=your_supabase_service_role_key
GEMINI_API_KEY=your_gemini_api_key
```

### 3. 로컬 실행
```bash
uv run uvicorn main:app --reload
```
실행이 완료되면 `http://127.0.0.1:8000/docs`에서 Swagger UI를 통해 API 테스트가 가능합니다.

## 🔌 API 명세서

### 오늘 자 모든 인터뷰 요약 가져오기
- **Endpoint**: `GET /today-summaries`
- **Query Parameters**:
  - `date` (선택 사항, `YYYY-MM-DD` 형식): 특정 날짜의 인터뷰 리스트를 가져옵니다. 미지정 시 한국 표준시(KST) 기준 오늘 날짜를 조회합니다.

#### 응답 예시 (JSON Array)
```json
[
  {
    "title": "2026년 6월 5일 (금) 임희윤 음악평론가, 박희아 음악평론가와의 인터뷰...",
    "content_url": "https://humblefactory.co.kr/11609/2026년-6월-5일-금-임희윤-음악평론가...",
    "summary": "요청하신 인터뷰 내용을 빈티지하고 정갈한 문체로 정리해 드립니다.\n\n### **[핵심 요약 3줄]**\n1. **글로벌 위상 강화...**\n...",
    "publish_date": "2026-06-05"
  }
]
```

## 🚀 Render 배포 가이드

[Render](https://render.com)는 FastAPI 애플리케이션을 손쉽게 배포할 수 있는 클라우드 서비스입니다.

### 1. 배포 설정
Render에 로그인한 뒤 **New Web Service**를 생성하고 본 저장소를 연동합니다.

- **Runtime**: `Python`
- **Build Command**: `pip install -r pyproject.toml` 또는 `pip install uv && uv pip install -r pyproject.toml`
  > 💡 **uv를 통한 빠른 빌드 커맨드 설정:**
  > ```bash
  > pip install uv && uv pip install -r pyproject.toml --system
  > ```
- **Start Command**:
  ```bash
  uvicorn main:app --host 0.0.0.0 --port $PORT
  ```

### 2. Environment Variables (환경 변수 등록)
Render 웹 서비스 대시보드의 **Environment** 탭에서 로컬 `.env`에 정의된 변수들을 등록해 줍니다:
- `SUPABASE_PROJECT_URL`
- `SUPABASE_SECRET_KEY`
- `GEMINI_API_KEY`
- `PYTHON_VERSION`: `3.12.9` (사용하는 파이썬 버전 지정)

배포 완료 시 Render에서 기본으로 제공하는 퍼블릭 URL을 통해 뉴스 요약 서비스를 상용 배포로 이용할 수 있습니다.
